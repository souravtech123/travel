# 🌍 ExploreX Travel Website

🔗 **Live Demo**: [Click to view the project](https://tripza-travel.vercel.app/)

A responsive travel website built using **HTML**, **CSS**, and **JavaScript**. It features beautiful destination packages, a sticky navbar with smooth scroll navigation, a working contact form design, and an interactive image slider showcasing travel locations.

---

## ✨ Features

- 🌐 Clean landing page with travel theme
- 📦 Travel packages section (cards with images and details)
- 🧭 Sticky navigation bar with smooth scroll
- 🖼️ Image slider for featured destinations
- 📞 Contact section
- 📱 Responsive design for all devices

---

## ⚙️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
